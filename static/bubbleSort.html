Ordenamiento de Burbuja

Acerca de

El ordenamiento de burbuja es un algoritmo simple utilizado para ordenar elementos en una lista. Funciona comparando pares de elementos adyacentes y realizando intercambios si están en el orden incorrecto. Este proceso se repite hasta que la lista esté completamente ordenada.

Duración:

La complejidad temporal del ordenamiento de burbuja es de O(n2), donde "n" representa el número de elementos en la lista a ordenar. En el peor caso, se requerirán n*(n-1)/2 comparaciones e intercambios.
